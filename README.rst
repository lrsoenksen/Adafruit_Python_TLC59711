
Introduction
============

Python module for the TLC59711 16-bit 12 channel LED PWM driver.

Dependencies
=============
This driver depends on:

* `py-spidev <https://github.com/doceme/py-spidev>`_

Usage Example
=============

See examples/simpletest.py for a demo of the usage.
